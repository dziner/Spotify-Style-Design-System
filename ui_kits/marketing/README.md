# Marketing — UI Kit

Spotify's marketing/account surfaces (Premium, signup, careers) are *lighter* and more editorial than the player. This kit shows two patterns:

- **Premium hero** — black background, oversized display type, green CTA
- **Plan comparison cards** — 2×2 grid of plan tiles with feature lists

Open `index.html` for the demo.

## Files
- `index.html` — assembled marketing page
- `MarketingNav.jsx` — top nav with green CTA
- `Hero.jsx` — display headline + CTA + key art panel
- `PlanCard.jsx` — pricing plan tile
- `Footer.jsx` — link columns + social

This is intentionally a small kit — the focus of this design system is the dark in-app experience.
